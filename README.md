# Graziella Fusari

Website institucional de arquitetura terapêutica para `graziellafusari.com`.

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- App Router
- Deploy preparado para Vercel

## Páginas

- `/` página inicial
- `/sobre` abordagem e conceito
- `/servicos` consultorias e limpeza energética
- `/processo` método e paleta sensorial
- `/contato` canais de contato

## Desenvolvimento local

Instale as dependências:

```bash
npm install
```

Rode o servidor de desenvolvimento:

```bash
npm run dev
```

Acesse:

```text
http://localhost:3000
```

## Build

Gere o build de produção:

```bash
npm run build
```

Rode o build localmente:

```bash
npm run start
```

## Deploy na Vercel

O projeto usa `next.config.ts` com configuração padrão para Vercel. Para publicar:

1. Conecte o repositório `graziberry/graziellafusari.com` na Vercel.
2. Use o framework preset `Next.js`.
3. Build command: `npm run build`.
4. Install command: `npm install`.
5. Output directory: padrão da Vercel para Next.js.

## Design

A interface segue uma direção visual suave e sensorial:

- Off-white, creme e areia como base
- Terracota para ação e calor
- Verde sálvia para biofilia e equilíbrio
- Marrom profundo para tipografia e contraste
- Bordas discretas, cards com raio de 8px e composição editorial

## Conteúdo

O site está em português brasileiro e apresenta a proposta de arquitetura terapêutica com foco em casa, pessoa, aroma, interiores sensoriais, organização e bem-estar.
